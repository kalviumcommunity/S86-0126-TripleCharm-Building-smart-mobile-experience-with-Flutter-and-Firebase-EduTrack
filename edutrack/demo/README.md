# Demo Screenshots

This folder contains screenshots demonstrating EduTrack's responsive design across different devices and orientations.

## Required Screenshots:

1. **phone_portrait.png** - Phone in portrait mode (example: Pixel 6)
2. **phone_landscape.png** - Phone in landscape mode
3. **tablet_portrait.png** - Tablet in portrait mode (example: iPad Pro)
4. **tablet_landscape.png** - Tablet in landscape mode
5. **welcome_screen.png** - Welcome screen screenshot

## How to Capture:

```bash
# Run the app
flutter run -d <device-id>

# Take screenshot using:
# - Emulator: Click camera icon in toolbar
# - Physical device: Use device screenshot function
# - Chrome: Right-click > Save image as...
```

## Naming Convention:
- Use lowercase with underscores
- Include device type and orientation
- Save as PNG format for best quality
