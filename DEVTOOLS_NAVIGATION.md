### Step 3: Launch App (5 minutes)
```bash
cd edutrack/
flutter run -d chrome
# Wait for: "DevTools is available at: http://..."
```

### Step 4: Try Hot Reload (10 minutes)
```
1. In dashboard_screen.dart, find AppBar title
2. Change: "Dashboard" ‚Üí "EduTrack Dashboard"
3. Save file: Ctrl+S
4. Press: r in terminal
5. Watch: Title updates instantly!
```

### Step 5: View Debug Logs (10 minutes)
```
1. Open Debug Console (View ‚Üí Debug Console)
2. Interact with app (click buttons, add items)
3. Watch: Console shows [DashboardScreen] logs
4. Read: What each log tells you
```

### Step 6: Open DevTools (5 minutes)
```
1. Copy URL from console: http://localhost:9100?uri=...
2. Paste into browser
3. Click: Widget Inspector tab
4. Click: Crosshair icon
5. Click: Widget in running app
6. View: Properties in right panel
```

### Step 7: Read Deep Guide (40 minutes)
```

```
